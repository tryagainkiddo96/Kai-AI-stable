# Kai All-in-One Control Center

## [v1.0.0] - 2026-05-02
- Initial stable release of the all-in-one Kai control center (CLI-based).
- Features:
  - CLI Control Panel with chat surface and 7-action quick dock
  - Cross-platform launchers (Windows, macOS, Linux) and desktop launcher
  - macOS launcher (.command) and Windows PowerShell/Bat launchers
  - kai_control_panel_run wrapper to simplify local runs
  - Lightweight demo flows (safe lab) and chess buddy integration
  - A small MCP-lite orchestrator for demonstration only
- How to install: clone repo, install dependencies, run panel via launcher or module
- Testing: quick smoke checks, readiness probes, and demo sequences

Future updates
- Web UI MVP (optional)
- More widgets (memory explorer, reports, incident logs)
- Authentication and audit trails for diverse environments
